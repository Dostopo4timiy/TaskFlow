# Асинхронный сервис управления задачами

Асинхронный микросервис для управления задачами с использованием FastAPI, PostgreSQL, RabbitMQ и Docker.

## Функциональные возможности

- Создание задач через REST API
- Асинхронная обработка задач в фоновом режиме
- Параллельная обработка нескольких задач
- Система приоритетов для задач
- Пагинация и фильтрация списка задач
- Мониторинг статуса выполнения задач
- Отмена выполняемых задач

## Технологический стек

- **Python 3.10+**
- **FastAPI** - асинхронный веб-фреймворк
- **PostgreSQL 14+** - база данных
- **RabbitMQ** - очередь сообщений
- **SQLAlchemy** - ORM
- **Alembic** - миграции базы данных
- **Docker** и **Docker Compose** - контейнеризация
- **Pytest** - тестирование
